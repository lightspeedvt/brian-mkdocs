Adding a link to your social media outlets

The Social Media Bar, once activated, will appear on the left-side of the screen on all pages in the System (if the screen size is too small, it will display the links at the bottom of the page, below any content) It is controlled by Access Level, and can be further controlled per System Theme.

Configuration

Step 1: Navigate to 401.1 Manage Systems. Select your System and then open the "System Feature Toggles" section. Activate the Social Media Bar using the "Show Social Media Bar" toggle. Once you select "yes" a new section will appear where you can select which access levels can view it.

Step 2: At the top of the screen select the "Themer Settings" tab, and then select which Theme to edit. After selecting a theme, you will see the Social Media Settings tab on the left of the screen.

Social Media Metadata options:

Title: What the title of the message should be (Example: Your System or Company name)

Description: A short sentence describing the link or System the user will be sharing.

URL to Share: Any URL you'd like to redirect the user to. (Example: the URL to the front door of your System where users can find out more information about you.)

Image URL (Minimum 200x200): Enter a path to an Image you'd like to use. This image is used by LinkedIn and Google Plus, but not Twitter or Facebook.

Twitter Message: This is an additional field where you can add @'s or #'s for your twitter message. (Please keep in mind Twitter's 140 character limit)

Share Buttons to Display:

Select which social sites to allow sharing to. Current options are: Facebook, LinkedIn, Google Plus, and Twitter. Here you can also select which side of the screen to display the buttons, on the left or right side.

Notes:

  1. You will need to reset your session if you make any changes to this social media section. Log out and log back in again for the changes to show up.

  2. If the Social Media Bar is turned on, but no values are entered at the Theme level, the social bar won't be displayed until there is data added.

  3. Facebook does not allow you to override the image of the page you are trying to share. So no matter what image you specify, it will always grab the image from the Facebook metadata from the destination URL.

Landing Page

If you'd like to use our custom shareable landing page, you can toggle the "Use Custom Landing Page" option to "Yes". If this setting is turned on, it will create a sharable public page that you can customize. (Note: this page can override the thumbnail image issue on Facebook)

Landing Page URL Format: Http://vt.lightspeedvt.com/share/?id=[themeid]

Note: Your Theme ID is different than your System ID. Your Theme ID is displayed in parenthesis near the top of the page, under the Theme Selection drop-down.

Redirect URL (Or not)

If you would like to override the metadata but still want the use this page to go to a page hosted elsewhere, this will allow that. Leave blank if you don't want to redirect.

The next two fields (share page image url and html/video embed code) are if you don't want to redirect to another page. The share page will display the title, description, and either an image or the html/video entered. If you embed a video, the image is not displayed.

Notes:

  1. This share page will use your System Theme information to add the background image and other elements to the page.

  2. The share page is responsive and is mobile friendly.

Call to Action Fields

You can add a Call to Action button to your landing page if you wish. The following settings are available:

Button Color: What color should this button be?
Button Text: What should the text on the button say?
Text Color: What color should the text be?
URL: What URL should we take the user to when clicked?
