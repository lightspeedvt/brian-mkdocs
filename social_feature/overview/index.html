Adding a link to your social media outlets

The Social Media Bar, once activated, will appear on the left-side of the screen on all pages in the System. It is controlled by Access Level, and further controlled per Theme.

Step 1: Activate the Social Media Bar under System Settings and select which access levels can view it.

Step 2: Access the Theme Editor for the System/Theme you want to edit and select the "Social Media" tab on the left.

Social Media Metadata options:

Title: What the title of the message should be (typically your System or Company name).

Description: A short sentence describing the link or System the user will be sharing.

URL to Share: Typically the URL to the front door of your System where users can find out more information about you.

Image URL (Minimum 200x200): This image is used by LinkedIn and Google Plus, but not Twitter or Facebook.

Twitter Message: This is an additional field where you can add @'s or #'s for your twitter message. (Please keep in mind Twitter's 140 character limit)

Share Buttons to Display:

Select which social sites to allow sharing to. Current options are: Facebook, LinkedIn, Google Plus, and Twitter.

Notes:

  1. You will need to reset your session if you make any changes to this social media section. Log out and log back in again for the changes to show up.

  2. If the Social Media Bar is turned on, but no values are entered at the Theme level, the social bar won't be displayed until there is data added.

  3. Facebook does not allow you to override the image of the page you are trying to share. So no matter what image you specify, it will always grab the image from the Facebook metadata from the destination URL.

Landing Page Section

If you'd like to use a custom share landing page hosted by LightSpeed VT, you can toggle "User Custom Landing Page" to "Yes". If this setting is turned on, it will share a public page that can override the image issue on facebook as well as carry over all the social media values above.

Landing Page URL Format: Http://vt.lightspeedvt.com/share/?id=[themeid]

Redirect URL

If a user wants to override the metadata but still wants the use to go to a page hossted elsewhere, this will allow that. Leave blank if you don't want to redirect.

Note: Using this redirect will show vt.lightspeedvt.com as the server. This can't be overridden. All other fields are honored.

The last two fields (share page image url and html/video embed code) are if the user doesn't want to redirect to another page. the share page will display the title, description, and either an image or the html/video enteed. if they embed a video, the image is not displayed.

the share page pulls the background color/image and font color of the theme ot use for the page.

The share page is responsive and is mobile friendly.

It will also show 'addthis ' buttons to share the page manually based on the social media values above.

Call to Action Fields

You can add a Call to Action button to your landing page if you wish. The following settings are available:

Button Color: What color should this button be? Button Text: What should the text on the button say? Text Color: What color should the text be? URL: What URL should we take the user to when clicked?
