  • Manage Unsubscribe List

Overview

If a user unsubscribes from any LSVT automated system email, you may remove them from the unsubscribe list (so you are subscribing them again) by accessing "400 - Application Administration" -> "409 - Vendor Tools"

Enter the user's email address and click "Go". If the user WAS unsubscribed, this will remove them from the list and they should start receiving emails again within 24 hours.
