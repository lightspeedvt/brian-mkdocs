Theme Settings

The “Themer” is a tool that allows you to manage and change all of the branding elements of a System, and a host of intricate settings that control how Locations and Users act.

Warning! When you create a new theme, it will automatically be applied to the location. This means you will overwrite the existing theme it was using. We recommend you CREATE A NEW LOCATION first, where you can build and test how your theme looks without affecting any live users.

Once your theme is ready to be turned on, it's really easy to apply it to live locations, and we explain those steps at the bottom of this article.

Creating a New Theme

First navigate to “403 - Theme Management” and then select your system. You will only need to select your system once, it will remember it on future visits.

Here you can either choose to edit your existing theme, or you can create new themes for your existing locations.

Part 1: Creating a New Theme

Upon clicking Create Theme the first thing you should notice is that you must hit SAVE before the theme will officially be created. We recommend simply hitting save and then continuing on with your work.

After you click SAVE the screen will update to system default colors - this is because as you update each field, you will see the colors and images appear in real time around the Themer.

Part 2: Menu Options and Naming your theme

In the top navigation we have three options:

  1. Navigate back to “Location Selection”
    • This is where you just came from, this section will allow you to select a different theme to manage.
  2. Them Editor (what we're in now)
    • This section controls the main settings for your theme.
  3. Components
    • This section controls the Main Menu Carousel and other Menus.

In the header area we can also change the name of our theme, and view how many locations this theme is currently assigned to. Clicking on the number (currently a “1” as we have not assigned this theme to any other locations yet) will take you back to “Location Selection” where it will display all locations using this theme.

Renaming your theme: If you are working with a lot of Themes it would be best to name this something unique - naming it “Theme 2” or something might be confusing down the line. For now we'll name our theme “VT System - Demo Theme” - which is the same name as our Location at the moment.

Part 3: Let's Get Themeing

The LightSpeed VT Themer is extremely powerful in that we provide a multitude of ways to manipulate the System look and feel. You will get a sense of all that you can accomplish as you read through this.

For each section below we will provide default imagery that you can use for your system. We suggest you build it out this way first to get a sense of each section and what you can do.

As you enter and update fields, you'll see that they turn GREEN to indicate a correct value has been entered and you can hit save. Entering an incorrect value will turn the field red and show an error warning on the left.
