The Themer

The Themer is a tool that allows you to manage and change all of the branding elements of your System, and a host of intricate settings that control how Locations and Users act.

Please be aware that any modification to your current "live" theme would affect all of your end users - if you have any questions at all about making a change, please feel free to contact us first before clicking that "save" button.

Warning!

If you are creating or cloning a new theme, you will want to create a NEW LOCATION first. When you Create or Clone a Theme, it is automatically applied to the location. This means you will overwrite the existing theme it was using - and if there are any Users at that location, they will see the changes immediately. CREATE A NEW LOCATION first, where you can build and test how your theme looks without affecting any live users.

After your Theme has been created to your liking, you can apply that theme to any live location that you desire.

Creating or Managing a Theme

To get to the Theme Manager, navigate to the Super User Dashboard and then to “403 - Theme Management.” Once inside, select your System (typically you will only have one System to choose from.)

Every System has a Main System Theme - that is the theme you are logged into right now. Now you can either choose to edit your existing main system theme, or you can create or clone new themes for your existing locations.

Part 1: Creating a New Theme (Or CLONE and existing Theme)

The first thing you may want to do to begin learning the Themer would be to CLONE your existing theme into a new location you have created exclusively for testing.

To clone your existing theme, click on the "Clone Theme" button next to your system theme. Once the dialogue box appears, select which location to clone this theme to. Remember to be careful to only clone this theme to your testing location for now - as you could overwrite a LIVE location with theme settings that are not intended.

Note

When you CLONE a theme, our system will duplicate every setting EXCEPT the Menus - once your theme is cloned and you would like to use it immediately, be sure to visit the Menus section and click "Auto Generate" on the menus so they will be added to your new Theme.

If you are cloning a theme rather than creating one from scratch - you should now be ready to go to make any tweaks you need to adjust the logo, colors, images, etc..

Creating a New Theme

The following sections below will cover in detail what each section of the themer will adjust - it includes "default" settings that are intended for someone to create a theme for the first time. The settings in the "default" fields will very likely be different than the ones you have existing now.

Upon clicking Create Theme the first thing you should notice is that you must hit SAVE before the theme will officially be created. We recommend simply hitting save and then continuing on with your work.

After you click SAVE the screen will update to system default colors - this is because as you update each field, you will see the colors and images appear in real time around the Themer.

Part 2: Menu Options and Naming your theme

In the top navigation we have three options:

  1. Navigate back to “Location Selection”
    • This is where you just came from, this section will allow you to select a different theme to manage.
  2. Them Editor (what we're in now)
    • This section controls the main settings for your theme.
  3. Components
    • This section controls the Main Menu Carousel and other Menus.

In the header area we can also change the name of our theme, and view how many locations this theme is currently assigned to. Clicking on the number (currently a “1” as we have not assigned this theme to any other locations yet) will take you back to “Location Selection” where it will display all locations using this theme.

Renaming your theme: If you are working with a lot of Themes it would be best to name this something unique - naming it “Theme 2” or something might be confusing down the line. For now we'll name our theme “VT System - Demo Theme” - which is the same name as our Location at the moment.

Components - Menus

In this section you can modify existing Menus (The top navigation bar, the hamburger menu, and the System Modules) or add new items to the menus.

When adding a new Menu item, we have the following available fields:

Name: What is the name of this Menu item?
URL: Where does this Menu item link to?
Target: Is this opening on the current page, or a new page?
Rank: What order should this be placed in amongst the other menu items?
*Icon: What icon would you like to use for this? (Optional)
Enabled: Turn this ON or OFF

*In the "icon" field, you can add an icon to your button. You can choose from our library of available icons found HERE. To add an icon, simply write in the word "icon-" and following the "-" enter the icon name. For example, for a Graduation cap, you would type in: "icon-graduation"
