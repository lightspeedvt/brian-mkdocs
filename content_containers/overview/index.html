Content Containers

We are currently working on a Super User Admin area for this feature, which will be available soon. In the meantime, please reach out to the Client Care department for assistance if you need anything.

Content Roles

Content Roles control what content is available for a user to see. This is a change to the prior LSVT designation where Access Levels used to control what content was visible to a user. Any user can be ser to as many, or as few, Content Roles as needed.

Content Roles can be created as needed per "System" - and all the Locations within the System will have these Content Roles available for users to be assigned to them.

A Course, or multiple Courses, can be assigned to a Content Role. This allows us to control what individual users can see, so it is very flexible.

Premier Content

A Premier Role is a Content Role that is not visible to be selected in the User's profile, but can be added by a Top Admin, or a transactional purchase from a Shopping Cart. This may be useful where there is "Premium Content" that can be purchased by indibidual users at a location, and by it being a Premier Content Role, a Manager at that location cannot assign this Premium content to a user without oversight by a Top Admin (Super User A & B), or a paid transaction.

The Premier Content Roles will also allow for you to hide desired Content Roles at desired Locations, as it is a "Per Location" Setting. What this means is that if you have some Content Roles that are only applicable for a certain number of Locations and not all Locations, this gives you the ability to "hide" these sensitive Content Roles as needed from individual Locations.

If you have access to Premier Content, you can view and modify this content in two areas:

  1. You can set a role to be "Premier" Under 402 - Content Containers. In that area you will see a “Premier Content” tab. From there you can select a “System” that you have access to, and then you will see that list of all Locations that you can update the Premier Content at. From here you can set the content roles to be Premier at one Location, but not Premier at another Location.

  2. The second area is under "Manage Users" - When you select a user to edit, you will see a section below the "Standard Content" section where you can add or remove "Premier" content.
